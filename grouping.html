Concept 2: Grouping

Primary Maths P3–P6  |  Learning Out of the Box

4 Steps to Solve Any Grouping Question

1

1 Group

Identify what is in one group — the simplest combination of items.

2

Group Value

Calculate the total value of one group by adding each item's value.

3

Number of Groups

Divide: Total ÷ Group Value

= Number of Groups
4

Solve

Use number of groups to find the specific item the question asks for.

Question 1 — Kim's Coin Box Standard Grouping
Click each step below to see it light up here
Kim has savings of $20.40 in her coin box. All coins are fifty-cent and twenty-cent coins. There are 3 times as many 50¢ coins as 20¢ coins. Find the number of twenty-cent coins.

Step-by-step working

1
1 Group
3 fifty-cent coins & 1 twenty-cent coin
(ratio 3 : 1 → simplest group)
2
Group Value
3 × $0.50 = $1.50
1 × $0.20 = $0.20
$1.50 + $0.20 = $1.70
3
Number of Groups
= $20.40 ÷ $1.70
= 12 groups
4
Solve
Each group contains 1 twenty-cent coin
= 12 × 1 = 12
Number of twenty-cent coins = 12
Question 2 — Lincoln's Notes
Click each step below to build the solution
Lincoln has $165 worth of $5 notes and $2 notes. He has five more $5 notes than $2 notes. How many $5 notes does Lincoln have?
Type 2 (Guess & Check):
When there is a difference ("5 more"), first remove the extra amount to make equal groups, then group normally.

Step-by-step working

1
Handle the difference first
5 extra $5 notes = 5 × $5 = $25
$165 − $25 = $140 remaining
2
1 Group
1 five-dollar note + 1 two-dollar note
(now equal numbers → 1 : 1 ratio)
3
Group Value
$5 + $2 = $7
4
Number of Groups
= $140 ÷ $7
= 20 groups
5
Solve
$5 notes from groups = 20 × 1 = 20
Add back 5 extra: 20 + 5 = 25
Number of $5 notes = 25
Question 3 — Wholesaler's Boxes Least Number
Click each step below to build the solution
A wholesaler receives a shipment of 6500 plastic files. He wants to pack them into boxes. Each box can contain 30 files. What is the least number of boxes that he needs to pack all the files?
Least Number concept:
To use the fewest boxes, fill each box to its maximum first. Any leftover files still need one more box — even if it's not full.

Step-by-step working

1
1 Group = 1 Box
Each box holds a maximum of 30 files
2
Number of Groups
= 6500 ÷ 30
= 216 remainder 2
3
Handle the remainder
216 full boxes + 2 leftover files
The 2 files still need 1 more box
216 + 1 = 217
4
Solve — Least boxes needed
= 216 + 1 = 217
Least number of boxes = 217
Question 4 — Jacky's Notes
Click each step below to see it light up here
Jacky has a number of $10, $5 and $2 notes in the ratio 2 : 1 : 5. The total value of all the dollar notes is $560. How many $2 notes does Jacky have?

Step-by-step working

1
1 Group
2 × $10 notes, 1 × $5 note, 5 × $2 notes
(ratio 2 : 1 : 5 → simplest group)
2
Group Value
2×$10 + 1×$5 + 5×$2
= $20 + $5 + $10 = $35
3
Number of Groups
= $560 ÷ $35
= 16 groups
4
Solve
Each group has 5 two-dollar notes
= 16 × 5 = 80
Number of $2 notes = 80
Question 5 — Cory's Chocolates
Click each step below to build the solution
Cory bought 58 chocolates. For every 3 chocolates he buys, he gets one more for free. If each chocolate costs $2, how much did Cory pay?
Key idea: 1 group = 3 bought + 1 free = 4 chocolates total, but you only pay for 3. Handle the remainder carefully — leftover chocolates were bought (no free deal).

Step-by-step working

1
1 Group
3 paid + 1 free = 4 chocolates
(smallest repeating unit)
2
Group Value
Only 3 paid chocolates cost money
3 × $2 = $6
3
Number of Groups
= 58 ÷ 4
= 14 remainder 2
4
Solve
14 groups × $6 = $84
2 leftover (bought) × $2 = $4
Total = $84 + $4 = $88
Amount Cory paid = $88
Question 6 — Postman's Parcels
Click each step below to build the solution
A postman is paid $9.50 for every parcel he delivers. If the parcel is slightly damaged, he is only paid $7.50. Last month he made $2608. For every 9 parcels he delivered, 2 were damaged. How many undamaged parcels did he deliver?

Step-by-step working

1
1 Group
7 undamaged + 2 damaged = 9 parcels
(ratio from question: 9 total, 2 damaged)
2
Group Value
7 × $9.50 = $66.50
2 × $7.50 = $15.00
$66.50 + $15.00 = $81.50
3
Number of Groups
= $2608 ÷ $81.50
= 32 groups
4
Solve
Each group has 7 undamaged parcels
= 32 × 7 = 224
Number of undamaged parcels = 224
Question 7 — Light Bulbs
Click each step below to build the solution
A total of 282 red and yellow light bulbs were lined along a fence. There were at least 4 red bulbs between any 2 yellow bulbs. What is the largest possible number of yellow bulbs?
Strategy: To maximise yellow bulbs, use the minimum number of red bulbs (exactly 4) between each yellow. This gives the repeating pattern Y R R R R.

Step-by-step working

1
1 Group = Y R R R R
1 yellow + 4 red = 5 bulbs per group
(use minimum 4 red to maximise yellow)
2
Number of Groups
= 282 ÷ 5
= 56 remainder 2
3
Handle the remainder
2 leftover bulbs follow the pattern: Y R
So remainder gives 1 more yellow bulb
4
Solve
56 groups → 56 yellow
Remainder → 1 more yellow
= 56 + 1 = 57
Largest number of yellow bulbs = 57
